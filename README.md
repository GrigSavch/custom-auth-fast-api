# Система аутентификации и авторизации

Основана модель RBAC


> Автор: Григорий Савченко

## Тех стек:

- **Python 3.11+**
**FastAPI** - создание REST API
- **PostgreSQL** - база данных
- **SQLAlchemy** - ORM
- **pydantic** - модели


1. Пользователь может иметь несколько ролей
2. Роль определяет доступ к бизнес-элементу
3. Каждый бизнес-элемент имеет детализированные пермишены:
   - read
   - read_all
   - create
   - update
   - update_all
   - delete
   - delete_all
4. Если разрешение *_all = True, доступ ко всем объектам
5. Если *_permission = True, доступ только к объектам, где owner_id = user.id
